开工版真实要做的功能范围（必须实现）:

1. OS 识别
   - Windows / Linux / macOS / WSL 识别
   - 权限状态识别
   - Windows 提权入口

2. 防火墙
   - 至少 Windows 实现
   - 只处理 discovery 阶段需要的 UDP 规则
   - ensure_rules()
   - cleanup_rules()

3. 组网 / discovery
   - UDP multicast 发送与接收
   - pairing 流程
   - discovery 成功后打印对端地址/结果
   - 暂时不进入 TCP runtime

4. 手动键入 / 打印文本
   - discovery 失败后允许用户手动输入 IP 或文本地址
   - supervisor 打印当前状态
   - scripts/print_status.py 打印平台与运行状态


给 Codex 的明确开发边界说明:

1. 当前版本是“开工版”，只实现：
   - OS/platform detection
   - privilege check / Windows elevation hook
   - firewall setup/cleanup
   - UDP multicast discovery
   - manual input fallback
   - text printing / status output

2. 不要实现：
   - TCP 会话
   - scheduler runtime
   - worker runtime
   - task dispatch
   - result aggregation
   - heartbeat
   - compute backend

3. 对于未进入当前范围的模块，只创建 skeleton：
   - 文件存在
   - 类/函数签名存在
   - docstring 说明职责
   - 用 NotImplementedError 或 pass
